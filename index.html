About

Jim Wise Golf Academy CRM

The Jim Wise Golf Academy CRM powered by EspoCRM offers a comprehensive solution tailored specifically for golf instructors and their clients. This CRM seamlessly integrates EspoCRM to provide a user-friendly interface and robust functionality.

Unified Data Management

Effortlessly manage all your golf academy data in one application with EspoCRM integration, ensuring a seamless experience for instructors.

Intuitive Interface

Easily navigate the CRM with a user-friendly interface, making it simple for instructors schedules and track progress.

Accessible Anywhere

Stay connected on the go with a fully responsive design, ensuring access to the CRM from any device, anytime, anywhere hosted from a Virtual Private Server.

Efficient Integration

Streamline communication and scheduling with seamless email and calendar integration, ensuring timely updates and enhanced collaboration between instructors and customers.

Scrum

Sprint Cycles

Sprint 1

Setup local instances of EspoCRM on team members machines using the Xampp Software
Configured our Jira Task Board for direction into our three sprint cycles
Customized the EspoCRM instances (and developed documentation) for:
• Company Logo and Theme
• Additional Instructor Accounts
• Programs & Lessons Datatables
• Students & Parents Datatables
• Navbar Customization
• Customizable Entity Fields

Sprint 2

Setup a hosted Virtual Private Server with EspoCRM using the LAMP Infrastructure (Linux, Apache, MySQL, and PHP)
Transferred our local customizations into the hosted EspoCRM instance
Adjusted the Apache Server Webpage to connect using HTTPS with SSL Certification
Initialized the VPS with the correct packages and developed a secure MySQL Database that connects to EspoCRM
Created additional documentation for Theresa in order to properly use EspoCRM
Began Data Sanitization to allow for quick and streamlined data processing into EspoCRM

Sprint 3

Gave access to the CRM to our client for initial impressions and testing
Addressed questions the client had from their testing and further customized the CRM with their feedback
Created Excel templates to give to the client so they can easily import data into the CRM
Finished up documentation/user guides
Transitioned to DevOps mode where we built bash scripts that:
• Install the CRM instance
• Backup the database in case the application crashes
• Reinstall the CRM from the lastest backup
